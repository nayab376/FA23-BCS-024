# NZ-MarketPlace

## Overview

Full-stack e-commerce application platform.
Created by Nayab Zaman.

## Stack

- **Node.js**: Environment
- **Package Manager**: pnpm
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Express API 
- **Database**: PostgreSQL
- **Language**: TypeScript

## Setup Instructions

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
